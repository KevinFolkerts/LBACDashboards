## One-time setup

1. **Clone this repo locally**
   ```
   cd C:\Users\kdfol\Documents
   git clone https://github.com/KevinFolkerts/LBACDashboards.git
   ```

2. **Enable GitHub Pages** (one click)
   - Repo Settings → Pages → Source: **Deploy from a branch** → Branch: **main** / root
   - Wait ~1 minute. Your site is at https://kevinfolkerts.github.io/LBACDashboards/

3. **Authenticate git for push**
   The first time you run publish.py, Windows may prompt for GitHub credentials. Use a Personal Access Token if 2FA is on (Settings → Developer settings → Tokens → "repo" scope).

## Publishing updates

After dropping new CSV exports into `C:\Users\kdfol\Documents\MyScouting Reports\Membership Totals\`:

```
cd C:\Users\kdfol\Documents\LBACDashboards
python publish.py
```

That's it. Within a minute the live site reflects the new data; recipients just refresh.

## Want it fully automatic?

Schedule publish.py to run nightly via Windows Task Scheduler:

1. Open Task Scheduler
2. Create Basic Task → "LBAC Dashboard Publish"
3. Trigger: Daily at 7 AM
4. Action: Start a program
   - Program: `python`
   - Arguments: `publish.py`
   - Start in: `C:\Users\kdfol\Documents\LBACDashboards`
5. Done. The site refreshes every morning if you've dropped in new CSVs.
